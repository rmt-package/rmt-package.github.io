基础函数

终端的核心基础函数，用于日常交互和基本操作

echo

用法

echo <字母/文本>

作用

输出指定的字母、数字或文本内容，原样返回输入的所有字符（包括空格）

示例

|---------$ echo HelloWorld
|HelloWorld
|---------$ echo 你好，RedMaple！
|你好，RedMaple！

quit

用法

quit

作用

安全退出 Red Maple Terminal 终端程序，关闭当前会话

示例

|---------$ quit
|终端已关闭

help

用法

help [<函数名>]

作用

查看指定函数的详细说明，若不指定函数名，则显示所有可用函数的列表

示例

|---------$ help command
|---------$ echo
|函数：echo
|用法：echo <字母/文本>
|作用：输出指定的字母、数字或文本内容

version

用法

version

作用

显示当前 Red Maple Terminal 的版本信息，包括版本号和发行版本

示例

|---------$ version
|Red Maple Terminal v8.0-MS
|发行日期：2025-07-17
|版本：支持 Windows

工具函数

提供各种实用工具功能，辅助开发和日常计算任务

pi

用法

pi <位数>

作用

计算并输出指定位数的圆周率（π）数值，位数参数需为正整数，最大支持20000位

示例

|---------$ pi 8
|3.14159265

ascii

用法

ascii <字符>

作用

将字符串转换为对应的 ASCII 码值（十进制），支持标准 ASCII 字符（0-127）

示例

|---------$ ascii A
|65
|---------$ ascii @
|64
|---------$ ascii 1@A
|49,64,65

encrypt/decrypt

用法

encrypt <加密算法> <加密字符串> <加密密钥（部分算法需要）>

作用

将字符串加密，支持MD5（输入md5），移位加密（输入sc，需要整数密钥），红枫加密（输入rmc，需要密钥，解密还需要盐值）

示例

|---------$ encrypt md5 hello
|MD5:5d41402abc4b2a76b9719d911017c592
|---------$ encrypt rmc hello world
|原始文本: hello |加密后: 1f0a1e000b1f0a1e000b0000000000000000000000 |盐值（解密需要）: 8fb959ea0198df83f5f92ba95187b3c9
|---------$ decrypt rmc 1f0a1e000b1f0a1e000b0000000000000000000000 8fb959ea0198df83f5f92ba95187b3c9 world
|解密文本:hello

writefile

用法

writefile <文本内容> <文件名>

作用

创建新文件并将指定文本内容写入其中，文件将保存在当前工作目录下。若文件已存在，将覆盖原有内容。

示例

|---------$ writefile Hello,World! test.txt

writech

用法

writech <文本内容>

作用

将普通文本转换为适合 `writefile` 函数使用的格式，自动处理特殊字符和转义序列

示例

|---------$ writech
|这是第一行文本
|这是第二行文本
|这是一行文本`这是第二行文本
|---------$ writefile 这是一行文本`这是第二行文本 test.txt
|文件已创建：test.txt
|写入内容：
|这是一行文本
|这是第二行文本

invmode

用法

invmode

作用

开启排查模式。排查模式会显示更详细的系统信息和命令执行过程，帮助诊断问题。

示例

|---------$ invmode

系统函数

与操作系统交互的函数，用于系统管理和控制

cmd

用法

cmd

作用

在当前系统中打开命令提示符窗口（Windows）

示例

|---------$ cmd
|正在启动命令提示符...

powershell

用法

powershell

作用

在 Windows 系统中打开 PowerShell 命令行工具，提供更强大的系统管理能力

示例

|---------$ powershell
|正在启动 PowerShell...

close pc

用法

close pc

作用

安全关闭当前计算机

示例

|---------$ close pc

restart pc

用法

restart pc

作用

安全重启当前计算机

示例

|---------$ restart pc
